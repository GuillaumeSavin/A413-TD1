<body onload="init()">

document.getElementById("titre")

-La propriété de l'objet représentant ma balise pour récupérer son texte est innerText ce qui donne :
document.getElementById("titre").innerText

-La méthode que j'ai utilisé pour récupérer l'objet représentant ma balise h2 est :
document.getElementsByTagName("h2")[0]

let liste_h2 = document.getElementsByTagName("h2")
let taille_liste = liste_h2.length

taille_liste % 2

Réponses 6.2 Les propriétés innerHTML, innerText, outerHTML, outerText et textContent

-innerText : renvoie toutes les chaînes de caractères à l'intérieur de l'élèment HTML

-innerHTML : renvoie l'ensemble du contenu HTML à l'intérieur de l'élèment HTML

-outerHTML : renvoie l'ensemble du contenu HTML en plus des tags, des attributs à l'intérieur de l'élèment HTML

-outerText : renvoie toutes les chaînes de caractères à l'intérieur de l'élèment HTML mais supprime l'ancien élément quand modification

-Internet explorer ne supporte pas toutes les propriétés
